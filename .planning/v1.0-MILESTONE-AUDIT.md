## Cross-Phase Integration

All 26 requirements verified as wired end-to-end by integration checker. Key integration paths:

- **Auth → Board → Filters:** `_auth.tsx` guard → `fetchTasks` loader → `useFilterState` + `applyFilters` → `TaskBoard` render
- **Create → Realtime → Push:** `createTask` → Supabase INSERT → realtime subscription → `setTasks` update; separately → webhook → Edge Function → Web Push → service worker notification
- **Screenshot → Create/Edit/View:** `uploadScreenshot` → Storage signed URL → `screenshot_url` in task record → `ScreenshotThumbnail` + `ImageLightbox` on row
- **Filters → Realtime:** pure `applyFilters` runs on every render over live `tasks` array — new realtime events automatically filtered
