  <what-built>
Complete AI screenshot processing feature:
- Header "+" button is now a dropdown with "Upload Screenshot" and "Create Manually"

- Task form opens immediately with skeleton shimmer fields during processing
- Claude Vision API extracts task data from KakaoTalk screenshots

- Screenshot preview collapsible above form (expanded desktop, collapsed mobile)
- Saved tasks with screenshots show image icon on task row
- Tapping the icon opens full-size lightbox with accessible DialogTitle
- Screenshots stored in private Supabase Storage bucket (authenticated reads only)
- Claude API call and Storage upload run in parallel for speed

- Verify Storage bucket was created: check Supabase Dashboard -> Storage -> should see `screenshots` bucket
